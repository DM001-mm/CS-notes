# 标准：
+ 中间传输数据 不出错
+ 接受方来得及接受数据
# 实现
这个主要是通过**滑动窗口**来实现的

因为 TCP是双边互通协议，所以假定  数据 从A->B

A 有一个 发送缓冲区 ,缓冲区里面 包含 发送窗口，有三个指针来维护 这个滑动窗口 这个数据结构
头指针 第一个字节 这里是后沿
中间指针 未发送但可发送的第一个字节
尾指针 最后一个字节的下一个字节 这里是前沿
## 逻辑过程:
A 中的某一个应用程序 把 字节流 放到 tcp的发送缓存
B 中的应用程序将 tcp的接受缓存->应用程序
