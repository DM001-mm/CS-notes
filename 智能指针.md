## 实例化
```
auto p=std::make_shared<Typename>(); // 推荐 ，因为不用直接接触new，而直接接触裸指针
推荐用法是 创建对象 + 创建shared_ptr 管理它
而且这个速度更快，字数更少，内存局部性更好
```

## 赋值
```
int* a;
p1=a;
int b;
p1 =&b;
p1=p2;
```

## 注意

1. 避免忘记交给 `shared_ptr`
2. 避免同一个裸指针被多个 `shared_ptr` 管理
	都以为自己是 最后一个，造成重复释放
## 常见用法

可以直接替代指针

# unique_ptr

|类型|所有权|能不能复制|适合场景|
|---|---|---|---|
|`unique_ptr`|独占拥有|不能复制，只能移动|一个对象只有一个主人|
|`shared_ptr`|共享拥有|可以复制|多个地方共同使用同一个对象|
